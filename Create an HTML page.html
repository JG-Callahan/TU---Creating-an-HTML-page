HTML

HTML stands for HyperText Markup Language. "Hypertext" refers to links that connect web pages to one another, either within a single website or between websites.
HTML describes the structure of a web page.
HTML has tree structure consisting of branches and nodes.

HTML elements

HTML elements are the building blocks of HTML pages.
An HTML element is set off from other text in a document by "tags".

HTML tags

Tags consist of the element name surrounded by "<" and ">".
HTML5, the current standard for HTML, has roughly 100 tags.

web browsers

Browsers do not display the HTML tags, but use them to interpret the content of the page.

  • text
  • media
  • interaction
  • markup
  • classes
Many browsers have default styles for the appearance of the document based on the meaning of the tag.
